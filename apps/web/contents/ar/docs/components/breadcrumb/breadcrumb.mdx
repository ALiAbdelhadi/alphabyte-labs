## Usage
<div className="not-prose space-y-6">
  <Pre>{`import {
    Breadcrumb,
    BreadcrumbItem,
    BreadcrumbLink,
    BreadcrumbList,
    BreadcrumbPage,
    BreadcrumbSeparator,
} from "@/components/ui/breadcrumb"`}</Pre>
  <Pre>{`<Breadcrumb>
    <BreadcrumbList>

## Usage

<div className="not-prose space-y-6">
  <Pre>{`import { Badge } from '@/components/ui/badge'`}</Pre>
  <Pre>{`<Badge variant="outline">Badge</Badge>`}</Pre>
</div>
## Tips 
Tip: You can use <MdxBadge>badgeVariants</MdxBadge> to create your own
badge component with different styles.
<div className="not-prose space-y-6">
  <Pre>{`import { Badge } from '@/components/ui/badge'`}</Pre>
  <Pre>{`<Link className={badgeVariants({ variant: "outline" })}>Badge</Link>`}</Pre>
</div>

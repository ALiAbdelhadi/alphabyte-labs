# مكون AlertDialog

مكون AlertDialog هو نافذة حوار تظهر فوق المحتوى الحالي لتقديم معلومات مهمة أو طلب قرار من المستخدم. يختلف عن Dialog العادي في أنه يتطلب تفاعلًا من المستخدم قبل أن يتمكن من العودة إلى التطبيق.

## الميزات

- **متغيرات متعددة**: افتراضي، تحذيري، نجاح، تنبيه
- **أحجام مختلفة**: صغير، افتراضي، كبير، كبير جدًا، كامل
- **مواضع مختلفة**: وسط (افتراضي)، أعلى، أسفل
- **خيار إغلاق عند النقر خارج الحوار**
- **تخصيص كامل للأزرار**: حجم، لون، نمط
- **دعم كامل للوصول (a11y)**
- **تحسينات للرسوم المتحركة**

## الاستخدام الأساسي

```tsx
import {
  AlertDialog,
  AlertDialogAction,
  AlertDialogCancel,
  AlertDialogContent,
  AlertDialogDescription,
  AlertDialogFooter,
  AlertDialogHeader,
  AlertDialogTitle,
  AlertDialogTrigger,
} from "@/components/ui/alert-dialog"

export function AlertDialogDemo() {
  return (
    <AlertDialog>
      <AlertDialogTrigger>فتح</AlertDialogTrigger>
      <AlertDialogContent>
        <AlertDialogHeader>
          <AlertDialogTitle>هل أنت متأكد؟</AlertDialogTitle>
          <AlertDialogDescription>
            هذا الإجراء لا يمكن التراجع عنه.
          </AlertDialogDescription>
        </AlertDialogHeader>
        <AlertDialogFooter>
          <AlertDialogCancel>إلغاء</AlertDialogCancel>
          <AlertDialogAction>متابعة</AlertDialogAction>
        </AlertDialogFooter>
      </AlertDialogContent>
    </AlertDialog>
  )
}
```

## المتغيرات

يمكنك استخدام المتغيرات المختلفة لتغيير مظهر الحوار:

```tsx
<AlertDialogContent variant="default">
  {/* محتوى الحوار */}
</AlertDialogContent>

<AlertDialogContent variant="destructive">
  {/* محتوى الحوار */}
</AlertDialogContent>

<AlertDialogContent variant="success">
  {/* محتوى الحوار */}
</AlertDialogContent>

<AlertDialogContent variant="warning">
  {/* محتوى الحوار */}
</AlertDialogContent>
```

## الأحجام

يمكنك تغيير حجم الحوار باستخدام خاصية `size`:

```tsx
<AlertDialogContent size="sm">
  {/* محتوى الحوار */}
</AlertDialogContent>

<AlertDialogContent size="default">
  {/* محتوى الحوار */}
</AlertDialogContent>

<AlertDialogContent size="lg">
  {/* محتوى الحوار */}
</AlertDialogContent>

<AlertDialogContent size="xl">
  {/* محتوى الحوار */}
</AlertDialogContent>

<AlertDialogContent size="full">
  {/* محتوى الحوار */}
</AlertDialogContent>
```

## المواضع

يمكنك تغيير موضع الحوار باستخدام خاصية `position`:

```tsx
<AlertDialogContent position="default">
  {/* محتوى الحوار */}
</AlertDialogContent>

<AlertDialogContent position="top">
  {/* محتوى الحوار */}
</AlertDialogContent>

<AlertDialogContent position="bottom">
  {/* محتوى الحوار */}
</AlertDialogContent>
```

## إغلاق الحوار عند النقر خارجه

يمكنك التحكم في إمكانية إغلاق الحوار عند النقر خارجه باستخدام خاصية `closeOnClickOutside`:

```tsx
<AlertDialogContent closeOnClickOutside={false}>
  {/* محتوى الحوار */}
</AlertDialogContent>
```

## تخصيص الأزرار

يمكنك تخصيص أزرار الحوار باستخدام خصائص `variant` و `size`:

```tsx
<AlertDialogFooter>
  <AlertDialogCancel variant="ghost" size="sm">
    إلغاء
  </AlertDialogCancel>
  <AlertDialogAction variant="default" size="sm">
    متابعة
  </AlertDialogAction>
</AlertDialogFooter>
```

## API

### AlertDialog

الحاوية الرئيسية للحوار.

### AlertDialogTrigger

العنصر الذي يفتح الحوار عند النقر عليه.

### AlertDialogContent

محتوى الحوار. يقبل الخصائص التالية:

- `variant`: نوع الحوار (`default`, `destructive`, `success`, `warning`)
- `size`: حجم الحوار (`sm`, `default`, `lg`, `xl`, `full`)
- `position`: موضع الحوار (`default`, `top`, `bottom`)
- `closeOnClickOutside`: إمكانية إغلاق الحوار عند النقر خارجه (افتراضي: `true`)

### AlertDialogHeader

حاوية لعنوان ووصف الحوار.

### AlertDialogFooter

حاوية لأزرار الحوار.

### AlertDialogTitle

عنوان الحوار.

### AlertDialogDescription

وصف الحوار.

### AlertDialogAction

زر الإجراء الرئيسي في الحوار. يقبل الخصائص التالية:

- `variant`: نوع الزر (جميع متغيرات Button)
- `size`: حجم الزر (جميع أحجام Button)

### AlertDialogCancel

زر إلغاء الحوار. يقبل الخصائص التالية:

- `variant`: نوع الزر (جميع متغيرات Button)
- `size`: حجم الزر (جميع أحجام Button)
